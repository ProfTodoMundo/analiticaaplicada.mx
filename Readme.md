# Analítica Aplicada MX

Sitio web oficial de **Analítica Aplicada MX**, un espacio de formación orientado a la reconversión laboral, el análisis de datos y el desarrollo de habilidades tecnológicas aplicadas al mundo profesional.

---

## 🌐 Objetivo del proyecto

Construir una plataforma digital que concentre:

- Información institucional
- Oferta académica
- Cursos y talleres
- Recursos visuales
- Reels y contenido educativo
- Semblanzas científicas
- Material de divulgación
- Formularios de contacto
- Acceso a redes sociales

---

## 🚀 Áreas principales del sitio

### 📊 Formación en análisis de datos
Cursos enfocados en:

- Python
- R
- Excel Profesional
- Power BI
- Git y GitHub
- Reconversión laboral tecnológica

---

### 🎥 Contenido multimedia
Integración de:

- TikTok
- Instagram Reels
- Facebook
- Flyers y carruseles
- Videos cortos educativos

---

### 🧠 Semblanzas científicas
Serie de contenido enfocada en:

- Científicos relevantes
- Historia de la ciencia
- Matemáticas
- Computación
- Inteligencia artificial
- Ciencia aplicada

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages
